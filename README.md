# React Pokedex


## What is React Pokedex?
 - React Pokedex is a simple Pokedex Web Interface using ReactJS and NodeJS
 
 
## Install:


  ```sh
  npm install
  ```
 or 
 ```sh
  yarn add
  ```
  
## Usage:


   ```sh
   npm start
   ```
  or
  ```sh
   yarn start
   ```
